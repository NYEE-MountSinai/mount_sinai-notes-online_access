MiDOC Flow Math

Given the MiDOC Output

How do we obtain the True Flow from the flow sensor and pressure sensors?

Given Variables

  • Time (in seconds)
  • Flow (in μL / min)
  • Pressure 3 (in mm Hg)
  • Pressure 4 (in mm Hg)

Output Variables

  • Flow through tube (in μL/s)
  • Total Volume Passthrough (in μL)

Method

Flow Calculation Method

  1. Convert Flow to μL/s via:
    flow(n) (in μL / min) / 60 = flow(n) (in μL / s)

  2. Instantaneous Trapezoidal Calculation of Volume from Flow
    Inst. Volume(n) (in μL) = (time(n) - time(n-1)) x (flow(n) + flow(n-1)) / 2

  3. Summation of Trapezoidal Volume
    Volume (in μL) = Inst. Volume(n) - Volume(n-1)

  4. Calculation of Flow / Volume Correspondence
    % = 1 - (Flow Sensor (μL) / Image Data (μL))

Flow / Pressure Drop Relationship

Zero the pressure sensors
Obtain Pressure difference in mm Hg
Convert Pressure difference to Pa
Multiply by

Microbore tubing measurement:
7 inches = 177.8mm
0.6mm radius

dP = 8 mu L Q / pi R^4
Where:
R = Radius of tube, ~0.6mm
L = Length of tube, = 1778 mm
mu = ?? ~1 [Joseph Sherwood]
Q = Flow
dP = difference in pressure sensors

Therefore:
Q = dP pi R^4 / 8 L mu
Q = dP * (2.862 418 503 × 10^(−13)) in SI
